# **App Name**: RideWise Splits by Spera Didiza

## Core Features:

- Terrain-Adjusted Split Calculation: Calculates race splits based on target time and course terrain, providing a detailed pacing strategy for the 947 Ride Joburg.
- Customizable Target Time: Allows users to input their target finish time in hours and minutes for personalized split calculations.
- Downloadable CSV: Generates and allows users to download a CSV file containing the calculated splits for offline use and analysis.
- Course Analysis: Presents key insights and pacing strategies based on historical data and terrain factors to help riders optimize their performance.
- Terrain Difficulty Indication: Visually represents terrain difficulty using icons to quickly identify challenging sections of the course.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a sense of strategy and forward momentum.
- Background color: Light blue-gray (#F9FAFB) for a clean, open feel.
- Accent color: Teal (#008080) to highlight actionable items and key data points.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable interface.
- Use simple, clear icons from Lucide to represent different terrains and actions (download, calculate, etc.).
- Use a clean, data-focused layout with clear sections for input, results, and course analysis.
- Subtle transition animations on calculations and data updates for a smooth user experience.
